    Logo Lab

    Generative tools for iconic logos

    Nine browser-based tools for generating minimalist marks. Each one trades on a narrow visual idea — displaced lines, interlocking rings, rounded-rect bitmaps — and gives you just enough knobs to find the good version. Every state has a shareable URL, every mark has a PNG/SVG export, and every tool embeds via iframe into anyone's site.

    Keyboard: press J/K to move between sections, 1–9 to jump, T to return to the top. Every embed is a live preview — click Open tool for the full controls and exports. Or skip the index and browse the Mosaic — a randomized wall of every tool.

    Embed URLs follow /logo/<slug>/#v=1&seed=<value> and stay stable across redesigns. Source on GitHub.
